## Quick start

## Setup
```bash
$ git clone https://gitlab.mynisum.com/dmantena/backbone-shopping--cart.git
$ cd backbone-shopping--cart
$ npm i
$ npm start
```

## Build
```bash
$ npm run prod:backbone
```

## Test 
```bash
$ npm run test
```
